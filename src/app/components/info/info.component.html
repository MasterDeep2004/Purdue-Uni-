FAQs

keyboard_arrow_right   What should I do if I am unable to create new entry?
  • Open Internet Explorer
  • Go to internet options
  • Go to security tab
  • Select trusted sites
  • Click on sites button
  • Add the website 'https://tools.library.pfw.edu/encyclopedia'
  • Close the trusted sites window
  • Select local intranet tab
  • Select custom level button
  • Scroll to the bottom and under UserAuthenticaion -> Logon choose 'Prompt for user name and password'
  • Select Ok
  • Apply these settings
  • Refresh the create entry page
 keyboard_arrow_right   Why can't I login?

The login is meant for PFW library administrators. If you want to contribute to the pages, please comment on the page and it will get addressed by the admin.

 keyboard_arrow_right   What type of files can I upload?

Currently the following file types are allowed to be uploaded :

  • Documents (.pdf)
  • Word Documents (.docx or .doc)
  • Sheets (.xlsx or .csv)
  • Powerpoint (.pptx)
  • Text (.txt)
  • Images (.png or .jpg)
  • Audio (.mp3)
  • Video (.mp4 or .mov or .flv or .mkv or .avi or .wmv or .webm)
 keyboard_arrow_right   Why can't I upload images directly when creating a page?

Uploading images directly converts the image into a text-like 'object', which is too large to save into a database. Hence if you try to do that, it will fail or break the system.

 keyboard_arrow_right   How do I attach tables to a page?

Tables are not currently supported by the rich text editor. To add a table, it's recommended to create a excel spreadsheet and upload it as an attachment.

 keyboard_arrow_right   Why can't I upload a large file?

Curently the size limit for uploading a file is 25 MB.

 keyboard_arrow_right   Do I need to login to comment or submit feedback?

Yes
